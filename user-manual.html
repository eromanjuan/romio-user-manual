ROMIO User Manual

ROMIO is an all-in-one workspace: a social feed, direct & group chat, collaborative workspaces with a no-code app builder, and personal tools. This guide walks through every feature.

info Overview

ROMIO brings several tools into one place. Use the left sidebar to move between areas. Each page has its own web address, so links are shareable and the browser back/forward buttons work.

Post text, photos, video, files, links, questions, notes and polls. Like, comment and @mention people.

Direct messages and automatic group chats for every workspace.

Role-based team spaces with a no-code app builder, dashboard tiles and an activity feed.

Personal Calendar, Checklist and Notes, plus a customizable widgets panel.

Light/dark, custom colors, frosted-glass cards and backgrounds — saved to your account.

login Getting started

Create an account

  1. On the sign-in screen, choose Create account.
  2. Enter your Surname, First name, a unique username (availability is checked as you type), your email, and a password (a strength meter guides you; confirm it).
  3. Submit. A verification email is sent — open it and verify before using the app.

Sign in

  • Use your email + password. Tick Remember me to stay signed in.
  • Google and Facebook sign-in buttons are available (these require the provider to be enabled in the project).
  • Forgot your password? Use the reset link to receive an email.
verified_user A green Verified badge appears once your email is confirmed. You can control whether it shows on your public profile (see Settings).

Mentions, likes & comments

  • Type @ to mention someone — pick from the list. Mentions are highlighted, clickable, and the person is notified.
  • Like and comment on any post; authors and thread participants get notified.
  • Click an author or commenter name to open their profile (your own name opens your editable profile).

Manage your posts

On your own post, the ⋯ menu offers:

  • Edit — change the text inline.
  • Hide — the post becomes visible only to you (unhide anytime).
  • Delete — asks for confirmation, then removes it for everyone.

chat Messages & chat

Open Work → Messages for a two-pane chat (conversation list + live thread; on phones the thread slides in).

Group chats

When a workspace is created, a group chat is made automatically with the owner and all members. As members are added or removed (or the workspace is renamed), the group chat updates. Group chats show the workspace's icon.

Direct messages

Start a 1-to-1 chat three ways:

  • Messages → New, then search a person.
  • From Search results, use the Message button.
  • From someone's profile, click Message.

DM avatars show the person's current profile photo. New messages raise the Messages badge in the sidebar and the notification bell.

account_circle Your profile

Profile photo (with editor)

  1. Go to My Profile and click the camera button on your avatar.
  2. Pick an image. In the editor you can drag to reposition, zoom (slider or wheel), rotate 90°, and flip horizontally/vertically. A circular guide shows the crop.
  3. Click Save. Your photo appears on your profile, the topbar and sidebar instantly. Use the trash button to remove it.

What others see (visibility)

In Settings → Profile visibility, toggle what visitors can see on your public profile:

  • Posts, Owned workspaces, Member since, Verified badge, Email.
visibility Visitors always see your name, username and photo; everything else follows your toggles.

notifications Notifications

The bell in the topbar shows an unread count and a dropdown. Clicking a notification marks it read and jumps to where it happened.

The sidebar also shows activity badges on Feed, Messages and Workspace. Visiting a section clears its badge.

search Search

Use the topbar search to find people, posts and workspaces:

  • People → open a profile or Message them.

dashboard Workspaces

A workspace is a team space with its own members, roles, dashboard and chat.

Create, open & switch

  • With no workspace yet, the Workspace page shows Create workspace. Give it a name, icon/color or an image.
  • In Settings → Select workspace, click Open on any workspace to switch to it (it becomes your current one). Changes appear instantly — no reload.

Members, roles & invites

  • Owners manage members from the workspace gear → Members: add people, set roles, remove, or bulk-change roles.
  • Roles: Owner (full), Editor (read+write), Viewer (read-only), or Custom permissions.
  • Invites and join requests are supported; owners approve/decline requests.

Deleting a workspace

delete_forever Deleting is protected: you must type the exact workspace name and enter your password twice (verified) before it's removed.

widgets Workspace builder (dashboard)

Inside a workspace is a full no-code dashboard (embedded builder). It has an activity feed, apps, and dashboard tiles.

Apps

  • You can import ready-made .rom.json app bundles.

build Creating a no-code app

An app is a custom data tool you build with no code — a task tracker, CRM, bug list, inventory, time log, anything. Each app is a set of records (rows), and you define the fields (columns) that make up a record.

1. Create the app

  1. In a workspace, click + Add app.
  2. Give it a name, and pick an icon and color.
  3. You can start from scratch, or import a template — a .rom.json bundle (several sample apps are provided: project tracker, bug tracker, sprint board, client CRM, time log, feature backlog).

2. Add fields

  1. Open the app and choose Add field.
  2. Pick a field type (see the Field types reference below) and give it a label.
  3. Configure the field's options where relevant:
    • Category / Status / Multi-select — add the choosable options, each with its own color.
    • Calculation — pick the number fields and the operation (sum, difference, product, etc.).
    • Days Between — choose the two date fields to measure between.
    • Auto-number — set a prefix (e.g. TASK-) for IDs like TASK-014.
    • Relationship — choose the target app whose records this field links to.
    • Number / Money / Rating / Progress — set ranges/format as offered.

3. Add & manage records

  • Click + Add (or the app tile's +) to create a record and fill in its fields.
  • Open any record to edit every field, add comments, or delete it.
  • Records show in a list/table; sort and browse with the built-in paging.

4. Reports & charts

Pin a Report to summarize an app — counts and charts grouped by a field (e.g. tasks by Status). Reports can be added to the dashboard as a Report tile.

5. Automations

Add simple when → then rules: when an item is created, updated, or a field changes to a value, then run an action (e.g. notify, set a field). Build multi-step actions with Add action.

upload_file Apps are portable — export an app as a .rom.json bundle and import it into another workspace.

construction Tools & widgets

Under Tools you get personal Calendar, Checklist and Notes — private to you and usable regardless of workspace membership.

Widgets panel (Feed & Profile)

The Feed and Profile show a personal Widgets panel. Click Add to choose from Clock, Date, Calendar, Calculator, Quick note, Checklist and Weather. In Edit mode you can drag to reorder and remove (✕). The Clock widget has a gear to pick its time zone.

settings Settings & theme

Each Settings section is a collapsible card — click the pencil to open it.

  • Profile — first name, surname, username (with availability), phone.
  • Profile visibility — what visitors can see.
  • Change email / Change password — reauthenticated for security.
  • Theme — see below.
  • Select workspace — add / open / set default / delete.

Theme

  • Dark / Light mode.
  • Custom palette: Accent, Background, Surface/cards, Text, Danger.
  • Card style: Solid or Glass (with Blur + Opacity sliders).
  • Background: None, a preset pattern (dots/grid/diagonal/glow/mesh) or an uploaded image.
  • Reset to default restores everything.
cloud_sync Your theme is saved to your account — log in on any device and it follows you.

smartphone Mobile & app

  • The whole app is responsive — the sidebar becomes a drawer on phones.
  • Install as a PWA: open the site in Chrome on Android → menu → Add to Home screen.
  • A native Android app (Capacitor) can be built from the project — see MOBILE.md.

quiz FAQ & troubleshooting

"Missing or insufficient permissions"

Usually means you're not signed in with the right account, or you're acting on something you don't own (for example, editing someone else's post, or a workspace you're only a viewer in).

My theme/photo isn't showing on another device

Hard-refresh after signing in. Themes are saved to your account; profile photos and backgrounds are stored inline and load with your profile.

A workspace shows apps I didn't add

Older workspaces may have inherited data before per-workspace isolation. Remove unwanted apps/tiles per workspace, or ask to reset that workspace.

memory Technical notes

  • Built on Firebase (Authentication + Cloud Firestore). Data updates in real time.
  • Firebase Storage is not enabled, so photos, small files and backgrounds are stored inline (compressed) with a size cap; large videos/files are shared by URL/embed instead of upload.
  • Weather uses Open-Meteo; icons use Tabler / Google Material Symbols.
  • Two tabs editing the same workspace builder at once is last-write-wins — edit in one place at a time.
